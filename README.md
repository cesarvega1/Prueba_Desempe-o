# 🎸 Aerosmith - Página Web Oficial 

## 📌 Descripción
Este proyecto es una página web estática dedicada a la legendaria banda de rock Aerosmith. Presenta información sobre la banda, eventos próximos, contenido multimedia y enlaces relevantes.

El objetivo es practicar desarrollo frontend utilizando HTML, CSS y JavaScript, creando una interfaz moderna y responsive.

---

## 🚀 Características
- ✅ Diseño moderno y responsive  
- ✅ Sección de eventos con tabla organizada  
- ✅ Menú de navegación con botón hamburguesa  
- ✅ Sección "About" con información de la banda  
- ✅ Galería multimedia (imágenes + video de YouTube)  
- ✅ Footer con enlaces y redes sociales
